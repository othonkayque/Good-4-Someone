# Good 4 Someone

Good 4 Someone é um projeto criado para alegrar o dia de alguém sem olhar a quem, você envia uma boa mensagem e recebe uma boa mensagem 

## Dependencias
 * Node 20.11.1
 * Express ^4.19.2
 * Prisma ^5.12.1
 * Tailwind ^3.4.3


### Comandos necessarios
`node main.js`.   
`npx prisma studio`.
